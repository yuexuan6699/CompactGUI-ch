# CompactGUI 中文汉化版

## 项目简介

CompactGUI 是一个 Windows 文件压缩工具的图形界面，使用 Windows 内置的 compact.exe 命令来压缩文件夹，节省磁盘空间。

## 汉化内容

本次汉化涵盖了以下内容：

### 界面翻译
1. **主窗口导航菜单**
   - Home → 首页
   - Watcher → 监视器
   - Compression DB → 压缩数据库
   - Settings → 设置

2. **首页界面**
   - select a folder → 选择文件夹
   - Add Folder to Queue → 添加文件夹到队列
   - Compress Selected → 压缩选中项
   - Version → 版本

3. **文件夹视图**
   - uncompressed size → 未压缩大小
   - contained files → 包含文件

4. **设置页面**
   - Filetype Management → 文件类型管理
   - System Integration → 系统集成
   - Compression Settings → 压缩设置
   - Background Watcher Settings → 后台监视器设置
   - Update Settings → 更新设置
   - UI Settings → 界面设置
   - 以及所有相关选项和提示

5. **压缩配置界面**
   - Compression Mode → 压缩模式
   - Configuration → 配置
   - Skip file types specified in settings → 跳过设置中指定的文件类型
   - Skip file types likely to compress poorly → 跳过可能压缩效果差的文件类型
   - Watch folder for changes → 监视文件夹变化
   - Apply to all → 应用到全部

6. **结果页面**
   - Compression Summary → 压缩摘要
   - Before → 压缩前
   - After → 压缩后
   - Space Saved → 节省空间
   - Files Compressed → 已压缩文件
   - Uncompress → 解压缩
   - Compress Again → 再次压缩
   - Submit Results → 提交结果

7. **数据库页面（暂时用不了已经隐藏）**
   - Database Results → 数据库结果
   - Search by game name or SteamID → 按游戏名称或 SteamID 搜索
   - Sort By → 排序方式
   - 排序选项（升序/降序等）

8. **压缩模式选择**
   - Estimated size → 预估大小
   - Savings → 节省
   - unknown → 未知

### 代码中的文本翻译
- 所有错误提示消息
- Snackbar 通知消息
- 文件夹验证错误信息

### 前提条件
- Windows 10/11 操作系统
- [.NET 9.0 Desktop Runtime](https://dotnet.microsoft.com/zh-cn/download/dotnet/thank-you/runtime-desktop-9.0.4-windows-x64-installer)
