# CompactGUI 中文汉化版

## 项目简介

CompactGUI 是一个 Windows 文件压缩工具的图形界面，使用 Windows 内置的 compact.exe 命令来压缩文件夹，节省磁盘空间。它可以透明压缩您的游戏和程序，在不影响其功能的情况下减少它们占用的空间。

## 功能特点

- **透明压缩**：文件可以在计算机上正常使用，就像什么都没发生一样，不需要解压缩即可直接使用
- **高压缩率**：游戏可压缩高达 60%（例如：ARK-生存进化 169 GB → 91.2 GB）
- **性能优化**：对于较旧 HDD 的用户甚至可能会看到加载时间缩短
- **多种压缩模式**：支持 XPRESS4K、XPRESS8K、XPRESS16K、LZX 等算法
- **后台监视器**：自动监控文件夹变化，保持压缩状态
- **社区数据库**：集成在线压缩预估数据库

## 使用方法

使用此工具可以在正常使用/运行文件夹的同时压缩它们：
- 减小游戏大小
- 减小程序大小（例如 Adobe Photoshop：1.71 GB > 886 MB）
- 压缩计算机上的任何其他文件夹

## 压缩选项

默认情况下，程序使用激活了 `XPRESS8K` 算法的 Compact 模式运行。可用的选项有：

| 模式 | 说明 |
|------|------|
| XPRESS4K | 最快，但压缩率最低 |
| XPRESS8K | 在速度和压缩率之间取得合理平衡 |
| XPRESS16K | 较慢，但压缩率更强 |
| LZX | 最慢，但压缩率最强 |

## 注意事项

1. **此工具不应用于在 Windows 11 上使用 DirectStorage 的游戏。** DirectStorage 允许游戏直接从 SSD 加载资源，绕过 CPU。压缩文件需要在发送到 GPU 之前解压缩，这将抵消任何性能提升。

## 什么是 compact.exe？

这是一个命令行工具，包含 Windows 10 引入的一系列新算法，可让您透明地压缩游戏、程序和其他文件夹，几乎没有性能损失。

### 透明压缩是什么意思？

透明压缩意味着文件可以在计算机上正常使用，就像什么都没发生一样——它们不会被重新打包成 Zip 和 Rar 文件。您仍然可以像以前一样浏览、启动游戏和程序。

### 这与旧版 Windows 内置的压缩功能有什么不同？

这类似于 Windows 内置的 NTFS-LZNT1 压缩（右键点击 > 属性 > 压缩以节省空间），但 Windows 10+ 引入的新算法更加先进，能够实现更高的压缩比，几乎不影响性能。
